What is MusiCode Lab?

MusiCode Lab is a music coding playground for young coders. You write programs using colorful blocks — and your program plays as real music!

Drag blocks from the left panel into the middle. Watch the code appear on the right. Hit ▶ Run to hear it play!

play( variable1 )
play( variable2 )

Music Phrases = Variables

Each variable stores a music phrase — a list of notes. Like a variable in Python: variable1 = [E4, G4, A4]

Click the edit button on a variable chip to change its notes. Try adding: C4 E4 G4 C5

def variable1 = [E4 G4 A4 G4 E4]

Loops = Repeat Blocks

The repeat block is like a for loop in Python. Drag play(variable1) inside it to repeat it 3 times!

repeat 3 times:
play( variable1 )
end

More Blocks

Explore all the block categories:

set tempo( 140 )
play note( C4, 1 beat )
say( "Hello!" )

You can also upload or record your own sound effects as audio variables!

Challenges & Downloads

Try Read Code mode to answer questions about code, or Listen & Code to build programs that match a target sound.

When you're done, hit Save to download your program as JSON, or Export to download the audio!

Press Ctrl+Enter to run/stop anytime. Happy coding!
